Black Friday | Web Animation

A one-page website narrating the story of the Starkid Musical: "Black Friday". Created as Multimedia Elective - Web Animation midterm and final course requirement.

SKILLS

Web Design, Web Development, Web Animation, Graphic Design, Layout, and Digital Illustration

TOOLBOX

Adobe Photoshop, Illustrator, & Dreamweaver, Visual Studio Code, and GitHub

Illustrations

All illustrations showcased on this website are digitally created by me and were featured in the animations. Some are animated in a parallax motion, while others are utilized as sprite-sheets. Originally envisioned by Nick Lang, I've reimagined the elements, including illustrated versions of the show's actors.

Embedded Website

While the website showcases screen-size specific computed animations, users can still enjoy its features through this embedded version. Now, they can interact with various animation techniques, ensuring an engaging experience regardless of their screen resolution.
